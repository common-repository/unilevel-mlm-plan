== Installation ==
<b>Follow the steps for automatic installation-</b>

– Log in to your WordPress dashboard
– Navigate to the Plugins menu and click Add New
– In the search field type Unilevel MLM Plan and click Search Plugins
– click on install 

– then activate Unilevel MLM Plan plugin
You are done :)
Manual Installation-
Place the unilevel-mlm-plan folder in your /wp-content/plugins/ directory.
Activate Unilevel MLM Plan.
A new menu item would be created in the admin menu called Unilevel MLM Plan.
